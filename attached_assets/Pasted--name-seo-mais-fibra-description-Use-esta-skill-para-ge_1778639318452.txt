## Templates de Meta Tags por Página

### REGRAS FIXAS
- `<title>`: máximo **60 caracteres** | formato: `[Palavra-chave] | Provider Mais Fibra`
- `<meta description>`: máximo **155 caracteres** | conter CTA + palavra-chave + diferencial
- `canonical`: sempre URL absoluta `https://www.providermaisfiibra.com.br[/path]`
- `og:image`: 1200×630px, WebP ou JPG, alt descritivo

## Sitemap XML (servidor Node.js)

```ts
// server/routes/seo.ts
import { Router } from 'express'
const router = Router()
const BASE = 'https://www.providermaisfiibra.com.br'

const PAGES = [
  { url: '/',              priority: '1.0', freq: 'weekly' },
  { url: '/contato',       priority: '0.9', freq: 'monthly' },
  { url: '/sobre-nos',     priority: '0.8', freq: 'monthly' },
  { url: '/onde-estamos',  priority: '0.9', freq: 'monthly' },
]

router.get('/sitemap.xml', (_req, res) => {
  const urls = PAGES.map(p => `
  <url>
    <loc>${BASE}${p.url}</loc>
    <changefreq>${p.freq}</changefreq>
    <priority>${p.priority}</priority>
  </url>`).join('')

  res.set('Content-Type', 'application/xml')
  res.send(`<?xml version="1.0" encoding="UTF-8"?>
<urlset xmlns="http://www.sitemaps.org/schemas/sitemap/0.9">
${urls}
</urlset>`)
})

router.get('/robots.txt', (_req, res) => {
  res.type('text/plain')
  res.send(
`User-agent: *
Allow: /
Disallow: /admin
Disallow: /api/

Sitemap: ${BASE}/sitemap.xml`
  )
})

export default router
```

## SEO On-Page: Guia de Conteúdo Semântico por Seção

### Hero (Home)
- `<h1>`: **"Internet Fibra Óptica no Oeste da Bahia"** — contém keyword primária
- Subtítulo `<p>`: mencionar velocidade, cidades, IPTV e CTA WhatsApp
- Exemplo: *"Planos de 300 a 900 Mega com IPTV incluso. Atendemos Barreiras, LEM, Correntina e mais 9 cidades da Bahia."*

### Seção Planos
- `<h2>`: **"Planos de Internet Fibra com IPTV Incluso"**
- Cada card de plano: usar `<h3>` com o nome do plano
- Texto do card: listar benefícios com termos semânticos (velocidade, streaming, roteador)

### Seção IPTV / Canais
- `<h2>`: **"IPTV com Canais de Esporte e Entretenimento"**
- Listar canais disponíveis (aumenta relevância para "IPTV + cidade")

### Seção Diferenciais
- `<h2>`: **"Por que escolher a Provider Mais Fibra?"**
- Cada item: texto com palavra semântica (fibra óptica, latência, suporte 24h, Anatel)

### Seção Onde Atuamos
- `<h2>`: **"Cobertura de Fibra Óptica no Oeste da Bahia"**
- Listar todas as 12 cidades com links internos para `/onde-estamos#cidade`
- Esta seção é crítica para SEO local — nomear cada cidade explicitamente

### Seção Avaliações
- `<h2>`: **"O que nossos clientes dizem"**
- Usar `itemscope itemtype="https://schema.org/Review"` no HTML dos cards

### Seção Indique e Ganhe
- `<h2>`: **"Indique e Ganhe com a Provider Mais Fibra"**
- Texto explicativo breve com CTA para WhatsApp

### FAQ (Tire suas Dúvidas)
- `<h2>`: **"Perguntas Frequentes"**
- Cada pergunta: `<h3>` — garante hierarquia correta para Google
- Usar o FAQSchema gerado acima

### Rodapé
- Incluir: nome, endereços, telefones, cidades atendidas, CNPJ, menção Anatel
- Link interno para todas as páginas do site
- Link externo para Instagram com `rel="noopener"`

## Checklist de Validação SEO

Execute após implementar qualquer mudança:

### Meta Tags
- [ ] Cada página tem `<title>` único ≤ 60 chars
- [ ] Cada página tem `<meta description>` único ≤ 155 chars
- [ ] Cada página tem `<link rel="canonical">` apontando para si mesma
- [ ] Home tem todas as meta OG e Twitter Card

### Structured Data
- [ ] Organization na home ✓
- [ ] LocalBusiness em /contato e /onde-estamos ✓
- [ ] FAQPage na home (seção FAQ) ✓
- [ ] BreadcrumbList nas páginas internas ✓
- [ ] Validar em: https://search.google.com/test/rich-results

### Conteúdo
- [ ] Única `<h1>` por página com keyword primária
- [ ] Hierarquia h1 → h2 → h3 sem pular níveis
- [ ] Todas as `<img>` têm `alt` descritivo com contexto
- [ ] 12 cidades mencionadas explicitamente no texto da Home
- [ ] CTA para WhatsApp em pelo menos 3 seções

### Técnico
- [ ] `/sitemap.xml` retorna XML válido
- [ ] `/robots.txt` não bloqueia páginas importantes
- [ ] HTTPS em todas as URLs
- [ ] Sem links quebrados (verificar com Screaming Frog ou Ahrefs)
- [ ] Velocidade: rodar PageSpeed e garantir score ≥ 90 (ver guia OPTIMIZE_PAGESPEED_100.md)

### Local SEO
- [ ] Google Business Profile criado para cada unidade
- [ ] NAP (Nome, Endereço, Telefone) consistente em todo o site e GBP
- [ ] Cidades mencionadas em texto corrido (não apenas em listas)
